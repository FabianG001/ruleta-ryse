# 🎡 Ruleta Ryse

Aplicación web desarrollada para realizar sorteos de colaboradores mediante una ruleta dinámica.

## 🚀 Funcionalidades

- Validación de antigüedad (8 meses)
- Eliminación automática de ganadores
- Animación fluida con sonidos y efectos
- Modo pantalla completa
- Diseño responsive

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript (Canvas API)
